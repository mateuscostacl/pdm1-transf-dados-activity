**Nesta tarefa você deve enviar dados de uma Activity para outra.

Material de apoio: https://fagno.github.io/des-movel-i-ifto/activity/#_enviando_dados_para_outra_tela

Crie um App que leia o nome de uma pessoa e apresente este dado em outra Activity de nome "ResultadoActivity".**

Na Activity "ResultadoActivity" apresente este dado em um TextView. Crie também dois botões, na qual, permita apresentar uma mensagem de boas vindas "Olá NOME, seja bem vindo!" usando toast e AlertDialog.

Ao efetuar o click nos botões, deve apresentar a mensagem de boas vindas utilizando o recurso definido no botão.
